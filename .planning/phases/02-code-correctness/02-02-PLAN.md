Replace the `saveFile` function (currently lines 331-341) with the try/finally version below. A `downloadStarted` flag is set to `true` only after `browser.downloads.download()` resolves successfully. If it throws, `downloadStarted` remains `false` and the finally block revokes the URL immediately. On the success path, the finally block is a no-op and the existing 10-second setTimeout handles revocation so the browser download manager can read the blob.

Replace the entire `saveFile` function with:

```javascript
async function saveFile(blob, filename) {
  const url = URL.createObjectURL(blob);
  let downloadStarted = false;

  try {
    await browser.downloads.download({
      url: url,
      filename: filename,
      saveAs: true
    });
    downloadStarted = true;
    // Browser download manager needs the URL alive briefly; revoke after 10 s
    setTimeout(() => URL.revokeObjectURL(url), 10000);
  } finally {
    if (!downloadStarted) {
      // Exception thrown before or during download — revoke immediately
      URL.revokeObjectURL(url);
    }
  }
}
```

This is the exact replacement. No other changes to download.js in this task.

Replace the `getBookMetadata` function (currently lines 118-137) with an explicit two-attempt sequential structure. No recursion. Logic:

1. First attempt: fetch `${API_BASE}/api/v2/epubs/${identifier}/`
2. If response is ok: return parsed JSON
3. If response is not ok AND identifier contains `:book:`: substitute `:article:` and make a second fetch
4. If second attempt is ok: return parsed JSON
5. If second attempt is not ok: throw with the second response's status
6. If first attempt is not ok AND identifier does NOT contain `:book:`: throw with the first response's status

Replace the entire `getBookMetadata` function with:

```javascript
async function getBookMetadata(identifier, jwtToken) {
  const headers = {
    'Accept': 'application/json',
    'Authorization': `Bearer ${jwtToken}`
  };

  const firstUrl = `${API_BASE}/api/v2/epubs/${identifier}/`;
  const firstResponse = await fetchWithRetry(firstUrl, { headers });

  if (firstResponse.ok) {
    return await firstResponse.json();
  }

  if (identifier.includes(':book:')) {
    const articleUrn = identifier.replace(':book:', ':article:');
    const secondUrl = `${API_BASE}/api/v2/epubs/${articleUrn}/`;
    const secondResponse = await fetchWithRetry(secondUrl, { headers });

    if (secondResponse.ok) {
      return await secondResponse.json();
    }
    throw new Error(`Failed to fetch metadata: ${secondResponse.status}`);
  }

  throw new Error(`Failed to fetch metadata: ${firstResponse.status}`);
}
```
